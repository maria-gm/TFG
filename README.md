# TFG – Aplicación Web para Técnicas de Análisis Multivariante

Aplicación interactiva desarrollada con **R** y **Shiny** como parte del Trabajo Fin de Grado en Estadística.

## Descripción

La aplicación reúne distintas técnicas de análisis multivariante en una única interfaz, permitiendo:

- Cargar datos propios o utilizar un conjunto de ejemplo.
- Aplicar diferentes técnicas de análisis multivariante.
- Visualizar resultados mediante tablas y gráficos.
- Obtener una breve interpretación de los resultados para facilitar su comprensión.

Además, la aplicación realiza automáticamente tareas de preprocesado como la selección de variables numéricas, la gestión de valores perdidos y la estandarización cuando la técnica lo requiere.

## Técnicas implementadas

- Análisis de Componentes Principales (PCA)
- Análisis Factorial
- Biplot
- Clustering Jerárquico
- K-Means
- DBSCAN
- Regresión Lineal Múltiple
- Métodos de regularización y reducción
- Regresión logística
- LDA
- Árboles de clasificación
